# extodo
vue3.js 기반으로 to do 앱을 만들어보자.
[스스로 만들어보는 todo list](https://choichumji.tistory.com/141)

![image](https://user-images.githubusercontent.com/40600306/153546936-727c826d-d767-4077-bcd2-eeea96ebe411.png)


## 구현 기능
- 시간대별 할일 목록 추가
- 할일 목록 삭제
- 할일 목록 완료
